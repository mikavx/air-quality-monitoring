Detailed Description

Główne okno aplikacji monitorującej jakość powietrza.

Odpowiada za interfejs użytkownika, w tym wyświetlanie list stacji i sensorów, prezentację danych pomiarowych na wykresie i w polu tekstowym, obsługę filtrowania stacji, zapisu/odczytu danych do/z pliku JSON oraz wyświetlanie wyników prostej analizy danych. Komunikuje się z GiosApiClient w celu pobierania danych z sieci.
