Detailed Description

Odpowiada za komunikację z publicznym API GIOŚ PJP.

Pobiera dane o stacjach pomiarowych, sensorach na tych stacjach oraz historyczne dane pomiarowe z wybranych sensorów. Komunikacja odbywa się asynchronicznie za pomocą QNetworkAccessManager. Wyniki zwracane są poprzez sygnały. Klasa obsługuje również podstawowe błędy sieciowe oraz błędy parsowania odpowiedzi JSON.
